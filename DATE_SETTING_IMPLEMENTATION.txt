✅ PENGATURAN TANGGAL DITERAPKAN KE SEMUA SISTEM BILLING
============================================================

SUMMARY PERUBAHAN:
==================

1. config.php (DIPERBARUI)
   - Tambah getCurrentDateTimeObject() → DateTime object dengan custom date
   - Tambah formatCustomDate() → Format tanggal dengan custom date support
   - Semua fungsi ini otomatis membaca nilai dari cookie app_custom_date dan app_auto_sync

2. generate_tagihan.php (DIPERBARUI)
   - generateTagihan() → Sekarang gunakan getCurrentDateTimeObject() untuk menghitung tanggal jatuh tempo
   - updateStatusPelanggan() → Sekarang gunakan getCurrentDateTimeObject() untuk menghitung keterlambatan pembayaran
   - Perhitungan sisa hari di halaman → Sekarang gunakan getCurrentDateTimeObject()
   - Perhitungan "Jatuh Tempo 3 Hari" → Sekarang gunakan formatCustomDate()

3. tagihan.php (DIPERBARUI)
   - Perhitungan "Jatuh Tempo 3 Hari" → Sekarang gunakan formatCustomDate()
   - Perhitungan sisa hari tagihan → Sekarang gunakan getCurrentDateTimeObject()

4. header.php (SUDAH ADA)
   - Tanggal di header sudah bisa diklik dan terbuka ke date_setting.php

5. date_setting.php (SUDAH ADA)
   - Menu untuk mengatur tanggal manual, automatic sync, dan timezone

CARA KERJANYA:
==============

Saat Anda mengubah tanggal di date_setting.php:
1. Pengaturan disimpan di COOKIE browser (app_custom_date, app_auto_sync, app_timezone)
2. Cookie ini otomatis dibaca oleh semua file PHP melalui config.php
3. Karena semua file billing sudah include config.php, maka mereka otomatis akan:
   - Menggunakan custom date untuk perhitungan (jika mode manual)
   - Menggunakan sistem date untuk perhitungan (jika mode auto sync)

DAMPAK:
=======

✅ Perhitungan tanggal jatuh tempo tagihan
✅ Perhitungan keterlambatan pembayaran (active/limit/inactive)
✅ Perhitungan sisa hari pembayaran
✅ Perhitungan notifikasi "Jatuh Tempo 3 Hari"
✅ Pembuatan tagihan otomatis
✅ Semua berdasarkan tanggal yang Anda set di date_setting.php

TESTING:
========

Untuk test, ikuti langkah:
1. Buka dashboard.php
2. Klik tanggal di header → masuk ke date_setting.php
3. Ubah tanggal ke yang Anda inginkan (misal: 2026-03-20)
4. Lihat perubahan di:
   - Halaman tagihan → sisa hari akan berubah
   - Status pelanggan → akan berubah based on new date
